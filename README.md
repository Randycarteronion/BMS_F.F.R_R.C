# CawBMS - 锂电池管理系统

## 🕹 简介

CawBMS 是一套基于 STM32F103 和 BQ76920 的锂电池管理系统。

所有设计资料基于 MIT 协议开源，欢迎大家复刻
## 🛠 功能

| 名称     | 开发状态 | 测试状态 | 名称      | 开发状态 | 测试状态 |
| ------ | ---- | ---- | ------- | ---- | ---- |
| 开机     | ✔    | ✔    | 欠压保护    | ✔    | ✔    |
| 关机     | ✔    | ✔    | 单节电压监测  | ✔    | ✔    |
| 初始化    | ✔    | ✔    | 电池组电压监测 | ✔    | ✔    |
| 短路保护   | ✔    | ❌    | 电流监测    | ✔    | ✔    |
| 过流保护   | ✔    | ✔    | SOC     | ❌    | ❌    |
| 过压保护   | ✔    | ✔    | SOH     | ❌    | ❌    |
| 充电被动均衡 | ✔    | ✔    | 上位机     | ❌    | ❌    |
